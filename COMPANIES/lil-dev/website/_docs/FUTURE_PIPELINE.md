# LIL DEV — FUTURE PIPELINE
*Locked opportunities to execute in sequence as the franchise grows*
*Version 1.0 — May 2026*

These are not ideas. They are locked intentions — confirmed directions for this franchise that will be executed when the timing, resources, or infrastructure is right. Each item has been evaluated and fits the core philosophy of building LilDev as a long-term IP ecosystem.

---

## PIPELINE CATEGORY 1 — ENTITY & DISCOVERY

- [ ] **Wikipedia / Wikidata entry** — after sufficient press coverage exists; establishes LilDev as a verifiable entity in the knowledge graph
- [ ] **Google Knowledge Panel claim** — after organic entity recognition appears; verify via Google
- [ ] **ISBN registration** — register each book as it's produced; instant Google Books cross-reference and entity authority
- [ ] **IMDB entry** — when animation launches; register character + creator
- [ ] **Google Books cross-reference** — follows from ISBN registration
- [ ] **Press kit page** (`/press/`) — media assets, brand story, character overview, creator bio; pitch to children's media outlets

- [ ] **Headless CMS** — when content volume exceeds static HTML management (Contentful, Sanity)
- [ ] **PWA (Progressive Web App)** — installable on mobile; app-like feel without app store submission
- [ ] **CDN optimization** — when global traffic warrants edge caching beyond Vercel's default
- [ ] **Dedicated media hosting** — when animation and video assets require more than GitHub storage

- [ ] **Analytics upgrade** — GA4 → full data warehouse when audience is large enough to warrant it

---

## SEQUENCING PHILOSOPHY

Do NOT attempt all of these at once. The sequence is:

1. **Foundation first** — website, entity signals, first store products
2. **Story second** — adventures published, characters established
3. **Media third** — YouTube, animated shorts, audio
4. **Games fourth** — browser games, then app
5. **Commerce fifth** — store fully populated, first sellout drop
6. **Community sixth** — newsletter strong, fan systems built
7. **Scale seventh** — international, licensing, retail

Each layer must be strong before building on top of it. A franchise built on a weak foundation collapses. Build the foundation like you're building for 25 years.
