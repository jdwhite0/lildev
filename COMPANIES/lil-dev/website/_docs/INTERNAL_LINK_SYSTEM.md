# LIL DEV — INTERNAL LINK SYSTEM
*Version 1.0 — May 2026*

---

## PHILOSOPHY

Internal links are the nervous system of the LilDev universe.
Every page should feel like it belongs in a living world — not a disconnected sitemap.

Links should serve three masters simultaneously:
1. **The user** — guide them deeper into the world naturally
2. **Google** — pass authority through meaningful anchor text
3. **The entity** — reinforce what LilDev IS at every junction

---

## THE AUTHORITY FLOW MODEL

```
HOME (lildev.world)
├── Meet Lil Dev ←→ Characters ←→ Angel
│   └── → Adventures, The World, Collectibles
├── The World ←→ World Map ←→ Lore
│   └── → Adventures, Characters, Origin Story
├── Adventures ←→ Books ←→ Animation
│   └── → Meet Lil Dev, Games, Store
├── Learn Lab ←→ Games ←→ Create Space
│   └── → Adventures, Parents
├── Parents ←→ Newsletter ←→ Community
│   └── → Learn Lab, Adventures, The World
└── Store (lildev.store) ←→ Collectibles ←→ Updates
    └── → Adventures, Characters, World
```

No orphan pages. Every page receives links from at least 2 others.
Every page links out to at least 2 others.

## ANCHOR TEXT STRATEGY

### Use Character Language (Good)
- "Meet Lil Dev"
- "Explore the World of Lil Dev"
- "Follow Lil Dev's Adventures"
- "Collect the LilDev Universe"
- "Enter the Learn Lab"
- "Angel's Story"
- "Get the LilDev Plush"

### Never Use (Bad)
- "Click here"
- "Read more"
- "Learn more"
- "This page"
- "Visit our store"

### Keyword-Rich Anchors (SEO Priority)
- "kids educational character" → `/parents/`
- "children's animated series" → `/adventures/`
- "Lil Dev collectible figures" → `lildev.store/collections/`
- "STEAM learning for kids" → `/learn/`
- "kids adventure books" → `/books/`

## CROSS-SITE LINK RULES (lildev.world ↔ lildev.store)

1. Every `lildev.store` product page includes: "Part of the [Adventure Name] Universe" → link to `/adventures/[adventure]/`
2. Every collectible product includes: "Meet the character" → `/meet-lil-dev/` or `/angel/`
3. Every `lildev.world` page with product mentions includes: "Available at LilDev Store" → `lildev.store`
4. Use `rel="noopener"` on cross-domain links but do NOT use `rel="nofollow"` — these are owned properties, authority should flow freely
5. Store homepage always has header link: "← Back to LilDev World" → `https://lildev.world`

## ORPHAN PAGE PREVENTION CHECKLIST

Before any page goes live, verify:
- [ ] Page appears in global navigation OR footer OR both
- [ ] At least 2 other pages link to it by URL
- [ ] Page links out to at least 2 other pages by URL
- [ ] Page is in `sitemap.xml`
- [ ] Page has a canonical `<link rel="canonical">` tag
- [ ] Page appears in the Updates/News feed if it's new content
